Privacy Policy

Saudi Exchange MCP — operated by Digitrends
Last updated: April 2026

1. Overview

This privacy policy explains how the Saudi Exchange MCP connector handles data when accessed through Claude.

2. Data We Collect

We do not collect, store, or retain any personal data. Specifically:

  • No user queries are logged or stored
  • No personal information is collected
  • No usage history is retained after a session ends
  • Authentication is handled entirely by Auth0 — we do not store passwords or tokens

3. Data We Access

When you use this connector, we access only:

  • The stock symbol or company name you ask about (to fetch data)
  • Your authentication token (validated in real-time, never stored)

4. Third-Party Services

This connector retrieves stock data from Yahoo Finance in real-time. No user data is shared with Yahoo Finance — only the stock symbol requested is used in the query.

Authentication is managed by Auth0. Please refer to Auth0's Privacy Policy for details on how they handle authentication data.

5. Data Storage

Our server is stateless. No database is used. No data is written to disk beyond what is required to serve the current request.

6. Data Transmission

All data is transmitted over HTTPS/TLS encrypted connections.

7. Your Rights

Since we do not store any personal data, there is no personal data to access, correct, or delete. For authentication-related data, please contact Auth0 directly.

8. Changes to This Policy

We may update this policy from time to time. The latest version will always be available at this URL.
